# Session Summary

## 概述

本技能用于对当前 OpenCode 会话进行总结，提取可复用的经验、方法和最佳实践。

## 使用方法

### 步骤 1：读取会话消息

使用 `session_read` 工具读取当前会话的所有消息：

```
session_id: 当前会话 ID
include_todos: true
include_transcript: true
```

### 步骤 2：分析会话内容

分析消息内容，提取以下信息：

1. **完成的工作** - 本次会话完成了哪些任务，解决了什么问题
2. **使用的工具和方法** - 使用了哪些工具、技能、代理、代码模式
3. **学到的经验** - 有哪些可复用的方法、技巧、教训
4. **待改进的地方** - 有哪些可以优化的地方
5. **建议的后续行动** - 如果有的话

### 步骤 3：输出总结

以结构化的方式输出总结报告，包含：

- 会话概览（任务类型、数量、复杂度）
- 关键成果
- 工具/技能使用情况
- 可复用的经验和方法
- 改进建议
